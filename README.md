

# 📘 Artificial Intelligence Lab Projects

**Practical AI / ML / DL lab implementations built on real datasets with complete pipelines, preprocessing, scaling, model training, evaluation, visualization, and FastAPI model deployment.**

---

### 🚀 Overview

This repository contains a collection of hands-on Artificial Intelligence labs covering core Machine Learning and Deep Learning concepts. Each lab demonstrates the complete end-to-end workflow — from data preprocessing and model training to evaluation and API-based deployment using FastAPI.

The goal of this repository is to bridge the gap between **model development and real-world deployment**.

---

### 🧠 Labs Included

* 🏥 **Diabetes Diagnostic Tool** — Medical prediction using classification models
* 🏠 **House Price Predictor** — Regression model for real estate pricing
* 🌸 **Iris Species Classifier** — Multi-class classification (ML basics)
* 💰 **Loan Approval Prediction** — Financial risk classification
* 🦠 **Malaria Cell Classification** — CNN-based medical image classifier
* 💼 **Salary Predictor** — Regression analysis on income data

### 🌐 Model Deployment

Selected labs are deployed as REST APIs using **FastAPI**, allowing real-time predictions via HTTP requests.

Deployment workflow includes:

1. Model training in Jupyter Notebook
2. Model serialization (Pickle/Joblib)
3. FastAPI backend creation
4. API endpoint testing
5. Integration readiness for web/apps
