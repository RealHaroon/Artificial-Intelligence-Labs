

# 📘 Artificial Intelligence Labs

**Practical AI / ML / DL lab implementations built on real datasets with complete pipelines, preprocessing, scaling, model training, evaluation, and visualization.**

---

### 🚀 Overview

This repository contains a collection of hands-on Artificial Intelligence labs covering core Machine Learning and Deep Learning concepts. Each lab demonstrates the end-to-end workflow — from data preprocessing to model evaluation and deployment-ready outputs.

---

### 🧠 Labs Included

* 🏥 **Diabetes Diagnostic Tool** — Medical prediction using classification models
* 🏠 **House Price Predictor** — Regression model for real estate pricing
* 🌸 **Iris Species Classifier** — Multi-class classification (ML basics)
* 💰 **Loan Approval Prediction** — Financial risk classification
* 🦠 **Malaria Cell Classification** — CNN-based medical image classifier
* 💼 **Salary Predictor** — Regression analysis on income data

---

### ⚙️ Tech Stack

* Python
* NumPy, Pandas
* Matplotlib / Seaborn
* Scikit-learn
* TensorFlow / Keras (for DL labs)
* Jupyter Notebook
